Projects

ACraffle

ACraffle is an interactive discord bot used in over 1500 servers and has over 3000 users. It connects to MongoDB in order to store and access user data across servers and is hosted using Heroku Cloud Hosting. It has features that allow users to create a profile and also has globally synced events such as a shop that resets daily.

Pokemon Clone

I made a simple Pokemon style game using HTML and JavaScript in which the player can move around the map and battle another Pokemon! I generated the map using Tiled, created the sound effects using SFXR, and the music is from YouTube. The animation is handled using the gsap JS library and the audio is handled using the Howler JS library. Check it out using the links below!
